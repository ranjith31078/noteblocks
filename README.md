# noteblocks

Noteblocks is a self-hosted notes app built with Next.js.

## Installation

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open http://localhost:3000 in your browser.

## Run in Production

1. Build the app:

```bash
npm run build
```

2. Start the production server:

```bash
npm run start
```

3. Open http://localhost:3000 in your browser.
